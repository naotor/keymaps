# naotor's Lily58 Layout

## Setup

```sh
$ cd /path/to/qmk_firmware/
$ git clone https://github.com/naotor/keymaps.git
$ mv ./keymaps/lily58/ ./keyboards/lily58/keymaps/my-keymap-name/
$ make lily58:my-keymap-name
```

## Layout

 * ,-----------------------------------------.                    ,-----------------------------------------.
 * | ESC  |   1  |   2  |   3  |   4  |   5  |                    |   6  |   7  |   8  |   9  |   0  |  =   |
