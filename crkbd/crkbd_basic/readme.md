# naotor's Corne Keyboard Layout

## Setup

```sh
$ cd /path/to/qmk_firmware/
$ git clone https://github.com/naotor/keymaps.git
$ mv ./keymaps/crkbd/ ./keyboards/crkbd/keymaps/my-keymap-name/
$ make crkbd:my-keymap-name
```

## Layout

### Qwerty

```
/* QWERTY
 * ,-----------------------------------------.                    ,-----------------------------------------.
 * | Tab  |   Q  |   W  |   E  |   R  |   T  |                    |   Y  |   U  |   I  |   O  |   P  |  -   |
 * |------+------+------+------+------+------|                    |------+------+------+------+------+------|
 * |LCTRL |   A  |   S  |   D  |   F  |   G  |                    |   H  |   J  |   K  |   L  |   ;  |  '   |
 * |------+------+------+------+------+------|                    |------+------+------+------+------+------|
 * |S(ESC)|   Z  |   X  |   C  |   V  |   B  |                    |   N  |   M  |   ,  |   .  |   /  | S(\) |
 * `-----------------------------------------+-------.     ,------+-----------------------------------------'
 *                          | LAlt |LOWER | /Space  /       \Enter \  |RAISE |BackSP|
 *                          |      |      |/       /         \      \ |      |      |

 */
```
